# 📊 Prompt-to-Response Evolution Visualizer

An interactive **Generative AI analysis tool** that demonstrates how small changes in prompt structure influence the responses generated by a transformer language model.

The application automatically generates multiple **prompt variants**, produces **LLM responses**, and visualizes how those changes affect response characteristics such as **length, sentiment, tokens, and subjectivity**.

Built using **Streamlit, HuggingFace Transformers, and Plotly**, the project provides an intuitive interface to explore **prompt engineering effects in real time**.

---

#  Features

###Prompt Variant Generation

Starting from a single base prompt, the system automatically generates multiple variants including:

* 🟣 Original Prompt
* 🔵 Constraint-based Prompt
* 🟢 Example-guided Prompt
* 🟡 Tone-modified Prompt
* 🔴 Detail-enhanced Prompt

These variants simulate common **prompt engineering techniques** used in real-world GenAI applications.

---

### Response Generation using LLM

Responses are generated using the **FLAN-T5 Small transformer model** from HuggingFace.

The model processes each prompt variant and produces a corresponding response that is then analyzed.

---

### Prompt Evolution Tree

A **visual tree diagram** shows the relationship between:

```
Base Prompt
     │
     ├── Variant 1 → Response
     ├── Variant 2 → Response
     ├── Variant 3 → Response
     └── ...
```

This allows users to easily observe how different prompt styles influence model output.

---

### 📊 Response Analysis Metrics

Each generated response is analyzed using NLP metrics such as:

| Metric             | Description                      |
| ------------------ | -------------------------------- |
| Word Count         | Number of words in the response  |
| Token Count        | Tokenized length of the response |
| Sentiment Polarity | Emotional polarity (-1 to +1)    |
| Subjectivity       | Objective vs subjective tone     |

---

### 📈 Interactive Visualizations

The application includes multiple interactive charts:

* Response Length Comparison
* Sentiment Polarity Distribution
* Token Count Analysis
* Polarity vs Subjectivity Scatter Plot

All charts are built using **Plotly** for dynamic interaction.

---

### 💬 Response Comparison Panels

Responses are displayed in **side-by-side cards**, allowing quick comparison between variants.

Each card shows:

* Variant type
* Generated response
* Word count
* Token count
* Sentiment indicator

---

### 📥 Export Analysis Data

Users can download the full analysis as a **CSV file** for further research or experimentation.
